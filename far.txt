	---Работа с панелями---
TAB - смена активной панели;
Ctrl+U - поменять панели местами;
Ctrl+L - показать свойства каталога;
Ctrl+Q - быстро показать содержимое файла/показать информацию о каталоге;
Ctrl+I - меню фильтров отображения;
Ctrl+T - отобразить древо каталогов;
Ctrl+O - убрать/показать обе панели;
Ctrl+P - убрать/показать неактивную панель;
Ctrl+F1 - убрать/показать левую панель;
Ctrl+F2 - убрать/показать правую панель;
Alt+F1/F2 - сменить левую/правую панель (другой диск, списки процессов...);
Ctrl+N - показать короткие/длинные имена объектов;
Ctrl+R - обновить список объектов;
Ctrl+F5 - растянуть окно на весь экран;
	---Расположение панелей---
Ctrl+↓↑←→ - изменение пропорций панелей; 
	---Работа с метками---
Ins/Shift+↓↑ - утсановить/снять пометку объекта;
Gray+/Gray- - установить/снять пометки объектов удовлетворяющие условию;
	"s*.*" - выделить все файлы начинающиеся на символ s;
	"*i*;*e*" - выделить все файлы в именах которых содержится символы i и e;
Shift+Gray+/Shift+Gray- - установить пометки на все объекты/снять пометки со всех объектов (не учитывает каталоги);
Gray* - инвертировать пометку;
Ctrl+M - восстановить предыдущую пометку;
Ctrl+Ins - копировать имена помеченых объектов в буфер обмена;
	---Работа с файлами и папками---
F3 - просмотр и редактирование файла;
F5 - копирование объектов;
F6 - перемещение и переименование объектов;
	F6 -> Путь: "\2\*old" - перемещение объектов с заменой их расширений;
Shift+F4 - создание текстового файла;
F7 - создание папки;
F8 - перемещение объекта вкорзину;
Alt+Del - уничтожение объекта;
Alt+F7 - поиск объектов;
Alt(зажат)+символ - быстрый поиск;
Ctrl+PgUp/PgDn - перейти в каталог выше/ниже (перейти в помеченый каталог);
Ctrl+A - поменять атрибуты объекта;
	--Завершить работу---
F10 - завершить работу Far;
	---Истории---
Alt+F8 - показать историю команд;
	Shift+Del - удалить выделенную запись;
Alt+F11 - показать историю просмотра/редактирования файлов;
	Shift+Del - удалить выделенную запись;
Alt+F12 - показать историю папок;
	Shift+Del - удалить выделенную запись;
	---Спец. команды---
Ctrl+\ - быстрый переход в корневой каталог;
	---Сортировка---
Ctrl+F12 - показать меню сортировки объектов;
Ctrl+F3 - сортировка по имени;
Ctrl+F6 - сортировка по размеру;
Ctrl+0-9 - разные виды отображения списка объектов;
	---Работа с редактором текста---
Home/End - переместить курсор в начало/конец строки;
PgUp/PgDn - переместить курсор в начало/конец столбца;
Ctrl+Home/End - переместить курсор в начало/конец файла;
Backspace/Del - удалить символ слева/справа от курсора;
F2 - сохранить изменения;
Shift+F2 - сохранить файл под другим именем, если были внесены изменения;
Ctrl+F7 - замена всех вхождений слов на другое (ищет все символы после курсора);
Shift+←→ - выделить символ строки;
Shift+Alt+↓↑ - выделить символ столбца и  строки;
Ctrl+U - снять выделение;
Ctrl+Y - удалить строку;
Ctrl+K - удалить текст от курсора до конца строки;
Esc - выход из редактора;
F6 - переключение режимов редактирования/просмотра;
F7 - поиск;
Ctrl+Z - отмена последнего действия;
	---Работа с архивами---
Shift+F1 - создать архив;
Shift+F2 - распокавать архив;
	---Работа с процессами---
Ctrl+F5:
	PID - униальный номер процесса;
	Page File (СтрнФайл) - запрошенная память у системы;
	Mem Usage (Память) - сколько физической оперативной памяти процесс использует прямо сейчас;
	Privаte ... (Private Bytes) - объём приватной памяти, выделенной исключительно этому процессу;
	VM Size (Virtual Memory Size) - зарезервированная память;
	CPU Time - суммарное время, которое процессор потратил на этот процесс с момента запуска;
	Privileged Time - время, когда процесс выполнял код в режиме ядра (Драйверы, работа с диском, сетью);
	User Time - время, когда процесс выполнял свой собственный код в пользовательском режиме;
"Idle" - это не настоящая программа, а специальный системный счётчик;
