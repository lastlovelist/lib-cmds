	1---Работа с каталогами---
ls [list-список] - выводит список содержимого каталога;
	"ls -l путь" - вывод списка содержимого с доп. информацией;
pwd [print working directory - вывести рабочий каталог] - вывести рабочий каталог;
mkdir [make directory - создать каталог] - создать каталог {"mkdir путь/имя"};
cd [change directory - изменить каталог] - изменить текущий каталог {"cd путь"};
	---Работа с файлами и каталогами---
rm [remove - удалять] - удалить (ссылки на) файлы или каталоги {"rm путь"};
	"rm путь -r" - удалить папку;
cp [copy - копировать] - копировать файл или каталог {cp путь_источника путь_копии/имя};
	"cp cp путь_источника путь_копии/имя -r" - копировать папку со всем содержимым;
	---Работа с файлами---
touch - изменяет времена, ассоциированные с файлом {"touch путь"};
	"touch test.txt" - создать пустой файл (если такого файла не существует);
cat [concatenate - конкатенация, сцеплять] - создавать и объединять содержимое файлов, а также считать данные в указанном файле и отображать их на экране {"cat путь"};
	---Работа с экраном---
clear - очистить экран;
